Grid-templates

Едно от най-добрите неща на Css Grid е, че можем да играждаме шаблони. Те представляват: "grid-template-" правила, след което добавяме съответно "columns" за контрол над колоните и "rows" за редовете.

Шаблоните създават толкова редове/колони, колкото стоиности им се зададат.

В този случай, задаваме 2 стойности за редове и 3 стоиности за колони.

Всички познати единици се приемат като стоиности в това число px, rem, em, %, vh, vw, както и специалната за css grid "fr"

.wrapper {
	display: grid;
	grid-template-rows: 50px 100px;
	grid-template-columns: 160px 260px 320px;
}

Разстояние

CSS box module допълва CSS Grid, за да позволи елементите да бъдат подравнени по реда на оста на колоната.

justify-items и justify-self подравняване на елементи по реда, и align-items и align-self на елементи по колоната.

justify-items елементи и align-items се прилагат към мрежовия контейнер и поддържат следните стойности:

     
  • auto
  • normal
  • start
  • end
  • center
  • stretch (Използва се по подразбиране)
  • baseline
  • first baseline
  • last baseline

Тези правила могат да бъдат използвани за подредба и на цели групи като се използва "-content"

.wrapper {
	display: grid;
	grid-template-rows: 50px 100px;
	grid-template-columns: 160px 260px 320px;
	gap: 1vw;
	justify-items: center
}

Навигация

Навигационна горна лента.


Съдържа 3 елемента с грид структури.

1. Цялостният лейаут е разделен автоматично на колони.

2. Грид-а в логото е ползван само за позициониране.

3. Връзките са подравнени в дясно от самият лейаут. Като самите те също са само подравнени чрез align, кактои и логото.

#navigationBar {
	display: grid;
	min-height: 100px;
	grid-auto-flow: column;
	grid-gap: 1.5em;
}
	
#logo {
	display: grid;
	justify-content: start;
	align-content: center;
}
	
#navigationLinks {
	display: grid;
	justify-content: right;
	align-content: center;
	grid-auto-flow: column;
}

А колонките с връзки са разделени на елемента, посока column всяка от която автоматично заема по 1 fr (fractioanl unit)

Галерия

Css grid е особено подходящ за изреждане на много елементи. С комбинация от няколко функции, можем да изградим галерия само от 3 реда код!


1. minmax(min, max)

Е функционална обозначение, която определя диапазон на размера, по-голям или равен на min и по-малък или равен на max. Ако max е по-малък от min, тогава max се игнорира и функцията се третира като min. Като максимална стойност определя гъвкавия фактор на трасето. Той е невалиден като минимум.


2. repeat([positive-integer | auto-fill | auto-fit] , track-list)

Е функция която повтаря елементите деца. Целта ѝ е да даде бърз и лесен начин за повтаряне на много редове или колони според използвания шаблон.

img {
	width: 100%;
	height: auto;
}
.wrapper {
	display: grid;
	grid-template-columns: repeat(auto-fill, minmax(300px, 1fr));
	gap: 1em;
}
